# Hello World

A simple JavaScript hello world example for Node.js:

```javascript
// hello.js
function hello(name) {
  console.log("Hello, " + name + "!");
}

// Run the greeting
hello();
```

## Running the Example

```bash
node hello.js
```

**Output:**
```
Hello, World!
```

## Custom Greeting

```javascript
hello("Developer");  // Output: Hello, Developer!
```
